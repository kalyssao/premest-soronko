Build a Blog app using React with the following features
- Login in and Sign up
- A view with all blog posts (show a small part of the text and show the full text 
on clicking the component, or a ‘View Full Post’ button)
- Add a Blog post
- Should have a title, content, author, date of addition
- Should have a number of upvotes and downvotes w buttons to ++ and -- the vote count
- Should be editable and deletable
